# Safe Paws Part 1

## Scope

Part 1 focused on the HTML-only foundation of the Safe Paws website.

- Semantic HTML page structure
- Consistent multi-page navigation
- Accessibility basics
- Native HTML forms and validation

## Pages Implemented

- `index.html` (Home)
- `about.html` (About Safe Paws)
- `adoption.html` (Available pets + adoption form)
- `contact.html` (Contact details + contact form)
- `enquiry.html` (General enquiry form)
- `faq.html` (Expandable FAQ using `details` and `summary`)

## Key Features Delivered

- Semantic sections (`header`, `nav`, `main`, `section`, `article`, `footer`)
- Accessible navigation (`aria-current`, skip links)
- Native interactive HTML (`details` and `summary`)
- Structured forms (`fieldset`, `legend`)
- HTML5 validation (`required`, `email`, `tel`, `number`, `pattern`)
- Contact links using `mailto` and `tel`

## Outcome

Part 1 delivered a complete and accessible HTML baseline, ready for styling and responsiveness in Part 2.
